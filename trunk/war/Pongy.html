Play Pong in Google Wave with Pongy

Pongy is a 2 real-time player Pong game that runs as a gadget in Google Wave. You need to have a Google Wave account to use this gadget.

To use the Gadget you can use the Gadget extension installer or simply add the gadget to a new Wave and invite someone to play a game with you.

Add the Pongy gadget by via the extension installer

  • Go to top menu item: Debug > Extensions > Add extension installer...
  • Enter the following Extension manifest URL: http://playpongy.appspot.com/manifest.xml and Insert the manifest.
  • Now you should see the Pongy icon when you open a Wave, simply click the icon to insert the gadget.

Add the Pongy gadget to a new wave

Click on the Gadget Icon at the top of your Wave and enter the following url: http://playpongy.appspot.com/pongy/org.cobogw.pongy.client.Pongy.gadget.xml

For more information, source code and bugs reports: http://code.google.com/p/pongy/

The gadget is has some bugs, and game play is not optimal yet.
